# 极客万年历 (Geek Calendar)

一个使用 PyQt5 和 Lunar-Python 开发的极客暗黑风格桌面万年历应用。该程序不仅提供桌面悬浮时钟微件，还包含一个功能丰富的全年月历视图，并支持中国传统农历、干支纪年、节气、节日及黄历宜忌显示。

## 🌟 功能特性

### 1. 桌面微件 (Desktop Widget)
- 🖥️ **无边框悬浮窗口**：静默运行在桌面右下角，完全不影响日常工作。
- 🕒 **实时时钟**：默认显示极简的小时与分钟，并支持通过大窗口设置是否显示精确到秒。
- 📅 **综合日期信息**：公历日期、星期、农历日期及干支纪年。
- 🎉 **智能节假日提示**：自动识别当天的农历节日、公历节日及二十四节气并高亮显示（没有时自动隐藏）。
- 🖱️ **便捷交互**：
  - 左键按住：自由拖动位置。
  - 左键双击：呼出详细万年历大窗口。
  - 右键点击：呼出快捷菜单（包含退出功能）。

### 2. 系统托盘图标 (System Tray)
- 🔔 **任务栏常驻图标**：程序启动后会在 Windows 任务栏右下角生成一个**蓝白配色的日历图标**，图标上**实时显示当天的日期数字**。
- 🖱️ **托盘交互**：
  - 左键单击：一键切换桌面悬浮时钟的显示/隐藏。
  - 左键双击：直接打开万年历大窗口。
  - 右键单击：呼出功能菜单。
- 📋 **托盘菜单**：
  - 显示/隐藏 桌面时钟
  - 打开万年历
  - **☑ 开机自动启动** - 一键开启或关闭开机自启（基于 Windows 用户级注册表，无需管理员权限，干净可逆）
  - 关于
  - 退出

### 3. 万年历大窗口 (Calendar Window)
- 🎨 **极客暗黑风 UI 与多主题切换**：除了经典的 VS Code 极客黑，还提供了深空蓝、护眼绿、赛博紫、秋叶褐、极简白共 **6 套预设高级主题**。点击顶部“🎨 主题”按钮即可一键切换。
- ⏱️ **秒数显示控制**：通过顶部栏的 `☑️ 显示秒` 复选框，可以一键控制桌面时钟是否展示秒数。
- 📆 **全景月历视图**：显示整个月的公历与农历日期，支持快捷切换上/下月和上/下一年。
- 🎯 **快速定位**：一键“回到今天”。
- 📌 **高亮标注**：今天及选中的日期带有边框和背景高亮，不同类型的节日（传统节日、公历节日、节气）有独特的配色区分，并在不同主题下自适应色彩。
- 📖 **详情侧边栏**：点击网格中的任意一天，右侧面板即显示该日期的详细信息，包含：
  - 超大号当前日期。
  - 详细的公历、星期。
  - 农历及八字干支。
  - **传统黄历宜忌**：动态展示当天的“宜”与“忌”（最多各列举8项）。
- 🛡️ **防误触设计**：点击右上角关闭按钮，默认行为是“最小化/隐藏”，你可以随时通过双击桌面微件将其重新唤出，避免意外结束进程。

## 🛠️ 环境要求

- Python 3.7+
- Windows (推荐，开机自启与托盘图标功能仅在 Windows 上生效) / macOS / Linux

## 📦 依赖安装

请使用 `pip` 安装所需的第三方库：

```bash
pip install PyQt5 lunar-python
```

> **说明**：
> - `PyQt5`：用于构建具有高度定制能力和流畅体验的 GUI 界面。
> - `lunar-python`：强大且精准的中国农历/黄历算法库。

## 🚀 运行方法

### 方式 1：直接运行源码

在终端或命令行中执行：

```bash
python desktop_calendar.py
```

如果你希望程序在后台静默运行而不弹出控制台黑框（特别是在 Windows 下），建议使用 `pythonw` 运行，或者将脚本后缀改为 `.pyw` 后双击运行：

```bash
pythonw desktop_calendar.py
```

### 方式 2：打包成独立 EXE（推荐，无需 Python 环境）

项目已附带一键打包脚本 `build.bat`，**双击运行**即可自动完成：

1. 自动从国内镜像（阿里云/腾讯/清华）安装 `PyInstaller`、`PyQt5`、`lunar-python`。
2. 自动生成多尺寸 `icon.ico` 图标。
3. 自动调用 PyInstaller 打包出**单文件 EXE**。

打包完成后会在 `dist\` 目录得到 `GeekCalendar.exe`：

- 🖼️ 文件本身就显示为蓝色日历图标。
- 📦 单文件约 50MB，可拷贝到任何 Windows 电脑直接运行。
- 🚫 完全无需被使用电脑上预装 Python 或任何依赖。

## ⌨️ 交互说明

1. 启动后，桌面的右下角会出现一个深色的悬浮时钟，同时 Windows 任务栏托盘区会出现一个**蓝白色日历图标**。
2. **拖动**：鼠标左键按住悬浮时钟的任意位置进行拖动。
3. **打开详情**：双击悬浮时钟，或双击托盘图标，即可弹出极客万年历大窗口。
4. **切换主题与设置秒数**：在大日历顶部标题栏可以勾选是否显示秒，也可以点击“主题”按钮在 6 套精美配色之间切换。
5. **选择日期**：在万年历窗口中，点击网格中的任何数字，右侧面板会更新该日的农历和宜忌信息。
6. **隐藏窗口**：点击极客万年历窗口右上角的 `×` 按钮，会将大窗口隐藏回后台（程序仍在托盘运行）。
7. **开机自启**：右键托盘图标 → 点击 `开机自动启动`（再次点击可取消）。
8. **退出程序**：右键托盘图标 → 选择 `退出`，或右键桌面悬浮时钟 → `Exit / 退出`。

## 📁 项目文件结构

```
万年历/
├── desktop_calendar.py    # 主程序：桌面微件 + 大日历 + 托盘 + 自启
├── generate_icon.py       # 打包前生成 .ico 图标的辅助脚本
├── build.bat              # 一键打包脚本（Windows）
├── README.md              # 项目说明文档
└── dist/
    └── GeekCalendar.exe   # 打包后的独立可执行文件
```

## 🔒 安全说明

本项目代码完全开源，**无任何后门或数据收集行为**：

- ❌ 不进行任何网络通信。
- ❌ 不读取/写入任何用户文件（除生成图标外）。
- ❌ 不调用任何敏感系统 API。
- ✅ 开机自启功能仅修改 `HKEY_CURRENT_USER` 注册表，与微信、QQ 等正规软件实现完全一致，无需管理员权限。
- ✅ 所有代码均明文可读。
